Cohort Retention Analysis

Задача:
Оценить, возвращаются ли пользователи после первой покупки.

Что сделал:
- Определил первую покупку пользователя
- Разбил пользователей на когорты по месяцу
- Посчитал retention по месяцам жизни

Ключевые выводы:
- Основной отток происходит после первой покупки
- Большинство пользователей не возвращается
- Удержание не улучшается со временем

Что это значит:
- Продукт не формирует привычку
- Есть проблемы с удержанием

Рекомендации:
- Улучшить первый пользовательский опыт
- Работать с удержанием после первой покупки
- Анализировать источники трафика
